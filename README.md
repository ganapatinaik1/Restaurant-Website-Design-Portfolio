# Restaurant Website Portfolio

A curated collection of modern, responsive restaurant websites built to showcase frontend development, UI/UX design, and rapid web prototyping skills. These websites were created as part of a personal outreach initiative, where each design was customized to match the branding and style of a different restaurant.

> **Note:** These are concept/demo websites created for portfolio and outreach purposes. They are not official websites of the respective restaurants.

## 🎯 Project Goal

The objective of this repository is to demonstrate the ability to rapidly design and develop visually appealing business websites while maintaining responsive layouts and a consistent user experience.

Each website explores different branding styles, layouts, typography, and color palettes tailored to the identity of a restaurant.
